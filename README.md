# 拾光 · 投命（SillyTavern 第三方扩展）

可点击命盘、随机抽取世界和攻略对象，并在确认入世后自动把世界设定发送到当前聊天，触发 AI 生成正式开场。

## 功能

- 命盘旋转与随机投命
- 世界、身份、主线、死局及攻略对象档案
- 入世状态、好感、积分、寿元、商城与背包
- 点击「确认进入此世」后自动发送设定并触发 AI 开场
- 手机端弹窗适配
- 本地保存轮回状态

## 发布到 GitHub

1. 在 GitHub 新建一个公开仓库，仓库名建议：`shiguang-cast-fate`。
2. 将本压缩包解压。
3. 把解压后的所有文件直接上传到仓库根目录。
4. 确认仓库首页能直接看到 `manifest.json`、`index.js`、`style.css`、`panel.html`。
5. 云酒馆安装地址填写：

   `https://github.com/你的用户名/shiguang-cast-fate`

不要填写 ZIP 下载地址，也不要填写单个文件地址。

## 安装

在 SillyTavern / 云酒馆的「安装第三方扩展」中粘贴 GitHub 仓库网址，安装后刷新页面并启用「拾光 · 投命」。

打开名称或描述中包含「拾光」「投命」或「快穿攻略系统」的角色卡，右下角会出现「命」按钮。

## 云酒馆前提

云酒馆必须开放第三方 UI 扩展安装和 JavaScript 执行权限。若平台只允许导入角色卡，不允许安装第三方扩展，则无法实现点击、旋转和自动开场。

## 故障排查

- **安装失败**：确认仓库公开，且 `manifest.json` 位于根目录。
- **看不到“命”按钮**：确认当前角色卡名称或描述含「拾光 / 投命 / 快穿攻略系统」。
- **命盘能转但无法自动开场**：确认当前聊天可正常生成，并且云酒馆没有禁用 STscript 的 `/send`、`/trigger`。
- **页面空白**：停用后重新启用扩展，并强制刷新页面。
